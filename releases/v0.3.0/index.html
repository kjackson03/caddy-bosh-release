Usage

Upload the release tarball from the command line:

bosh upload-release --name="caddy" --version="0.6.0" \
  "https://s3-external-1.amazonaws.com/dpb587-caddy-bosh-release-us-east-1/artifacts/release/ab1be028371778b20449d725f8edf3aed3ce429a" \
  --sha1="sha256:13fe7a3be3c7a499519d8ee79d54c3e0fafac89533bd04e74a7d360157dae4a9"

Or reference the release from a manifest:

releases:
- name: "caddy"
  version: "0.6.0"
  url: "https://s3-external-1.amazonaws.com/dpb587-caddy-bosh-release-us-east-1/artifacts/release/ab1be028371778b20449d725f8edf3aed3ce429a"
